Bienvenue sur ce site exclusivement consacré à Pokémon !

Vous y retrouverez un Pokédex de Pokémon Jaune ainsi qu'une partie "Stratégie".

A noter que comme ce site se base exclusivement sur Pokémon Jaune, certaines informations ne sont pas valables pour les jeux actuels.


Voici les anciennes règles qui régissaient le jeu:

  • Les types Fée, Acier et Ténèbres n'existent pas ;
  • Le type Poison possédait une faiblesse au type Insecte ;
  • Le type Insecte possédait une faiblesse au type Poison ;
  • L'Attaque Spéciale et la Défense Spéciale était combinée en une seule statistique, appelée "Spécial" ;
  • Les IV et les talents n'existaient pas ;
  • Les attaques étaient physiques ou spéciales selon leur type (ex: Ultralaser était une attaque physique) ;
  • Les EV existaient mais avaient une importance moindre car ils étaient tous maxés lors des combats (aujourd'hui, chaque Pokémon dispose de 512 EV à repartir dans les 6 statistiques avec un maximum de 256 EV par statistiques).
